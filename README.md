## Setup

```sh
git clone https://github.com/GPT-Engineer-App/upload-box-powerpoint-clone.git
cd upload-box-powerpoint-clone
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
